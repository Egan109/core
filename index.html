Core

The complete core Java software for the Transitime real-time transit information project

Download .zip Download .tar.gz View on GitHub

The software is checked in and made public only so that it can be viewed. While it has been used in a test environment it still is lacking significant features and is not ready for deployment.

Join the transitime forum on googlegroups to learn more and discuss this project. See https://groups.google.com/d/forum/transitime to join the group.

Documentation

Currently the main source of documentation is the Javadocs . In the future additional information on system architecture, components, setting up a system, etc. will be available here.
